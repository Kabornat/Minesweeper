Консольная версия игры, есть 3 сложности

Правила стандартные, откопать все клетки на которых нет мин

Выяснить наличие мин на клетке можно благодаря цифрам которые обозначают количество мин вокруг

Есть реализация BFS (Breadth First Search) для рекурсивного открытия пустых соседних клеток
